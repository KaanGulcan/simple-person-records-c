# Person Age Recorder (C)

This is a simple C program that records people's names and ages
entered from the keyboard and saves them to a file.

## Features
- Reads person name and age from standard input
- Saves data to a text file
- Uses struct and file I/O in C

## Technologies
- C programming language
- File operations (fopen, fprintf,fclose)
- Structs
  
## Türkçe Açıklama
Bu proje, klavyeden girilen kişi adı ve yaş bilgisini
dosyaya kaydeden basit bir C programıdır.
Struct ve dosya işlemleri kullanılmıştır.
